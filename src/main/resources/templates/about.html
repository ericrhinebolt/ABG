About

ABG is a full stack Java Spring Boot application built as a Capstone for a bootcamp. It utilizes HTML, CSS, JavaScript, Bootstrap, Thymeleaf, Java, Spring Security, Spring Data JPA and MariaDB.

The database schema is relatively basic. This project was built to display a full working CRUD application without being overly complex due to the time constraints of the project.

A few features I would implement given the time:

  • User profiles with the ability to change username and password.
  • Responsive star rating for reviews.
  • Adding Genres to games and extending the search functionality.
  • Adding scheduled API calls to update the games list.

Overall this project was a great learning experience. I plan on updating it and changing it as I learn more and newer technologies.
